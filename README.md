 SyncTech Ltd | Digital Engineering & Solutions

The official web platform for **SyncTech Ltd**, a premium digital agency specializing in custom software architecture, high-performance web development, and cloud infrastructure.

![Production Environment](https://synctechltd.space/og-image.png)

  Tech Stack
- Framework: [React 18](https://react.dev/)
- Build Tool: [Vite](https://vitejs.dev/) (with SWC for high-speed compilation)
- Styling: [Tailwind CSS](https://tailwindcss.com/)
- UI Architecture: [Shadcn/UI](https://ui.shadcn.com/) & [Radix UI](https://www.radix-ui.com/)
- Animations: Framer Motion (Optimized matte-glass interactions)
- Deployment: Hostinger via CI/CD GitHub Pipeline

  Key Features
- SPA Architecture: Smooth navigation via React Router DOM.
- Matte-Glass UI: Consistent glassmorphism design across all modules.
- SEO Optimized: Dynamic meta tags and OpenGraph support via React Helmet.
- CI/CD Integrated: Automatic production builds on every main branch push.
